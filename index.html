Acerca de este Portafolio

Este sitio web es nuestro portafolio de evidencias para la materia de “Producción Electrónica”, impartida por el Maestro Javier Osorio Figueroa.

Durante esta asignatura, nos enfocamos en el proceso completo para el desarrollo de proyectos electrónicos, incluyendo:

  • Diseño de circuitos esquemáticos.
  • Fabricación de placas electrónicas usando herramientas como "MonoFab" y "xTool".
  • Uso de software especializado "Altium" y "KiCad" para modelado y validación de diseños.

Herramientas de Trabajo

CNC
MonoFAB SRM-20

La fresadora de sobremesa monoFab SRM-20 es una máquina CNC de tres ejes diseñada para el prototipado sustractivo, ofreciendo un tamaño compacto y una funcionalidad potente a un precio asequible. Su gabinete totalmente cubierto reduce el polvo y el ruido, y cuenta con una ventana para visualizar el progreso del fresado. Un interruptor de seguridad detiene la operación al abrir la cubierta.

Corte Láser
Xtool Laser

Es un dispositivo de grabado y corte láser multifuncional que combina tecnologías de láser diodo e infrarrojo para cortar y grabar una amplia variedad de materiales con alta precisión. Su enfoque automático y posicionamiento visual permiten ajustes precisos para grabados detallados. Incorpora software compatible con múltiples sistemas operativos que facilita el diseño y control de los proyectos.
